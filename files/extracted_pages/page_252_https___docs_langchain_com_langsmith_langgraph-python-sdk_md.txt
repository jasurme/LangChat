LangGraph reference
===================

[![Image 5: PyPI - Version](https://img.shields.io/pypi/v/langgraph?label=%20)](https://pypi.org/project/langgraph/#history)[![Image 6: PyPI - License](https://img.shields.io/pypi/l/langgraph)](https://opensource.org/licenses/MIT)[![Image 7: PyPI - Downloads](https://img.shields.io/pepy/dt/langgraph)](https://pypistats.org/packages/langgraph)

Welcome to the LangGraph reference docs!

These pages detail the core interfaces you will use when building with LangGraph. Each section covers a different part of the ecosystem.

`langgraph`[¶](https://reference.langchain.com/python/langgraph/#langgraph "Copy anchor link to this section for reference")
----------------------------------------------------------------------------------------------------------------------------

The core APIs for the LangGraph open source library.

*   [Graphs](https://reference.langchain.com/python/langgraph/graphs/): Main graph abstraction and usage.
*   [Functional API](https://reference.langchain.com/python/langgraph/func/): Functional programming interface for graphs.
*   [Pregel](https://reference.langchain.com/python/langgraph/pregel/): Pregel-inspired computation model.
*   [Checkpointing](https://reference.langchain.com/python/langgraph/checkpoints/): Saving and restoring graph state.
*   [Storage](https://reference.langchain.com/python/langgraph/store/): Storage backends and options.
*   [Caching](https://reference.langchain.com/python/langgraph/cache/): Caching mechanisms for performance.
*   [Types](https://reference.langchain.com/python/langgraph/types/): Type definitions for graph components.
*   [Runtime](https://reference.langchain.com/python/langgraph/runtime/): Runtime configuration and options.
*   [Config](https://reference.langchain.com/python/langgraph/config/): Configuration options.
*   [Errors](https://reference.langchain.com/python/langgraph/errors/): Error types and handling.
*   [Constants](https://reference.langchain.com/python/langgraph/constants/): Global constants.
*   [Channels](https://reference.langchain.com/python/langgraph/channels/): Message passing and channels.

Model Context Protocol (MCP) support

To use MCP tools in your LangGraph application, check out [`langchain-mcp-adapters`](https://reference.langchain.com/python/langchain_mcp_adapters/).

*   [Supervisor](https://reference.langchain.com/python/langgraph/supervisor/): Orchestration and delegation.
*   [Swarm](https://reference.langchain.com/python/langgraph/swarm/): Multi-agent collaboration.
