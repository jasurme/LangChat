LangSmith SDK reference
=======================

[![Image 5: PyPI - Version](https://img.shields.io/pypi/v/langsmith?label=%20)](https://pypi.org/project/langsmith/#history)[![Image 6: PyPI - License](https://img.shields.io/pypi/l/langsmith)](https://opensource.org/licenses/MIT)[![Image 7: PyPI - Downloads](https://img.shields.io/pepy/dt/langsmith)](https://pypistats.org/packages/langsmith)

Welcome to the LangSmith Python SDK reference docs! These pages detail the core interfaces you will use when building with LangSmith's Observability and Evaluations tools.

For user guides, tutorials, and conceptual overviews, please visit the [LangSmith documentation](https://docs.langchain.com/langsmith/home).

--------------------------------------------------------------------------------------------------------------------------------------------------------

| Class/function | Description |
| --- | --- |
| [`Client`](https://reference.langchain.com/python/langsmith/observability/sdk/client/) | Synchronous client for interacting with the LangSmith API. |
| [`AsyncClient`](https://reference.langchain.com/python/langsmith/observability/sdk/async_client/) | Asynchronous client for interacting with the LangSmith API. |

Core APIs[¶](https://reference.langchain.com/python/langsmith/observability/sdk/#core-apis "Copy anchor link to this section for reference")
--------------------------------------------------------------------------------------------------------------------------------------------

The primary interfaces for the LangSmith SDK.

*   [`Client`](https://reference.langchain.com/python/langsmith/observability/sdk/client/): Synchronous client for the LangSmith API.
*   [`AsyncClient`](https://reference.langchain.com/python/langsmith/observability/sdk/async_client/): Asynchronous client for the LangSmith API.
*   [Run Helpers](https://reference.langchain.com/python/langsmith/observability/sdk/run_helpers/): Functions like `traceable`, `trace`, and tracing context management.
*   [Run Trees](https://reference.langchain.com/python/langsmith/observability/sdk/run_trees/): Tree structure for representing runs and nested runs.
*   [Evaluation](https://reference.langchain.com/python/langsmith/observability/sdk/evaluation/): Tools for evaluating functions and models on datasets.

Additional APIs[¶](https://reference.langchain.com/python/langsmith/observability/sdk/#additional-apis "Copy anchor link to this section for reference")
--------------------------------------------------------------------------------------------------------------------------------------------------------

*   [Schemas](https://reference.langchain.com/python/langsmith/observability/sdk/schemas/): Data schemas and type definitions.
*   [Utilities](https://reference.langchain.com/python/langsmith/observability/sdk/utils/): Utility classes including error types and thread pool executors.
*   [Wrappers](https://reference.langchain.com/python/langsmith/observability/sdk/wrappers/): Tracing wrappers for popular LLM providers.
*   [Anonymizer](https://reference.langchain.com/python/langsmith/observability/sdk/anonymizer/): Tools for anonymizing sensitive data.
*   [Testing](https://reference.langchain.com/python/langsmith/observability/sdk/testing/): Testing utilities and pytest integration.
*   [Expect API](https://reference.langchain.com/python/langsmith/observability/sdk/expect/): Assertions and expectations for testing.
